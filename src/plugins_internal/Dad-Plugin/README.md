# Dad Plugin

Yeah this on is simple, everytime someone starts a message with "Hi I'm . . .", the bot will make an epic dad joke.

## Config:

```json
    {
      "content": "I'm Dad."
    }
```

## Example:

> Someone: "Hi, I'm just a teenage dirtbag"

> Bot: "Hi just a teenage dirtbag, I'm Dad."
